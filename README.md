# shreygawde.com — Personal Portfolio Website

**shreygawde.com** is a personal portfolio website designed and developed by **Shrey Gawde**, combining modern web technologies with an experimental visual aesthetic.  
It serves as both a creative statement and a technical showcase.

##  Overview

The site embodies an “elegant psychedelic” design philosophy — merging minimal structure with vivid, reactive visuals.  
Each page features custom motion, color dynamics, and responsive layouts optimized for performance and accessibility.

##  Features

- Interactive parallax effects and hue-shifting gradients  
- Audio-reactive 3D visualizer built with Three.js  
- Responsive design with mobile-friendly layouts  
- Optimized asset loading and smooth transitions  
- Modular structure using React components  

## Tech Stack

- **React** (UI Framework)  
- **Vite** (Bundler / Dev Environment)  
- **TailwindCSS** (Styling)  
- **Three.js** (3D Visuals and Audio Reactivity)  

## Deployment

- Deployed via **Vercel** for optimal performance and continuous deployment.
